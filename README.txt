This program provides an interface for capturing images with one or more
Mightex CCD Cameras. It was built and tested using Mightex camera model
CGE-B013-U, but should work with other models with minimal to minor
adjustments. Configuration settings are loaded from the file `settings.txt`.

Depends on MSVC 2010 runtime (x86). Redistributable available here:
https://www.microsoft.com/en-us/download/details.aspx?id=5555

Mightex SDK and documentation available from:
http://www.mightexsystems.com/family_info.php?categories_id=141
